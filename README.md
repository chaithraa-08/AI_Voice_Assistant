# AI Voice Assistant

An AI-powered Voice Assistant built using Python and the Groq LLM. The assistant accepts voice input, understands natural language, and responds with speech output. It also integrates tools such as weather, calculator, web search, application launching, and camera control to provide an interactive and intelligent user experience.

## Project Status

🚧 Under Development (Approximately 60% Complete)

## Features Implemented

- 🎤 Voice Input (Speech-to-Text)
- 🗣️ Text-to-Speech Responses
- 🤖 AI Conversation using Groq LLM
- 💬 Conversation Memory
- 🌦️ Weather Information
- 🧮 Calculator Tool
- 🔍 Web Search
- 📂 Application Launcher
- 📷 Camera Integration
