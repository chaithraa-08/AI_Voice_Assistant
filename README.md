# 🎙️ AI Voice Assistant

An AI-powered Voice Assistant built using **Python** and the **Groq LLM** that enables natural voice-based interaction. The assistant accepts voice commands, understands user queries, and responds with realistic speech while providing useful tools such as weather updates, web search, calculator, application launching, and camera access.

> 🚧 **Project Status:** 60% Complete (Actively Under Development)

---

## ✨ Features

### ✅ Implemented

- 🎤 Voice Input (Speech-to-Text)
- 🤖 AI Chat using Groq LLM
- 🗣️ Text-to-Speech Responses
- 💬 Conversation Memory
- 🌍 Multi-language Detection
- 🌦️ Weather Information
- 🧮 Calculator Tool
- 🔍 Web Search
